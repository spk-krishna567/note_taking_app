# Note-taking-app

## React-Django Restful CRUD API with PostgreSQL

### notetaking folder -- django api (backend)

### react-notes folder -- react app (frontend)

#### api calls

```
1.Get    "api/notes"
2.Get    "api/notes/:id"
3.Post   "api/notes"
4.Put    "api/notes/:id/"
5.Delete "api/notes/:id"
6.Delete "api/notes"
7.Get    "api/notes?title=${title}"
```
